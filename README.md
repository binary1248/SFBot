SFBot
=============

What is SFBot?
----------------------

SFBot is, as its name implies a simple and fast IRC bot.

How do I use SFBot?
---------------------------

SFBot depends on:
- [SFNUL](https://github.com/binary1248/SFNUL)

Hack, build, run, enjoy.

What else can I do with SFBot?
--------------------------------------

The underlying bot framework was made to be as extensible as possible,
as such feel free to use it to do many other IRC-bot related things.
